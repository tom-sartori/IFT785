******** All commands available ******** 
 
help: Display this help information  <  
create_account: Add a new user  < [user_name]
add_block: Add a new block on ledger  <  
 For OpenBlock : [block_type], [owner_public_key], [unit], [balance], [minimal_balance]
 For SendBlock : [block_type], [owner_public_key], [receiver_public_key], [amount_to_send]
 For ReceiveBlock : [block_type], [receiver_public_key], [sender_public_key] 
show_account: Display the account of current user  < [public_key]
show_accounts: Display all accounts of ledger < 
show_blocks: Display all blocks of ledger  < 
show_ledger: Display the ledger  <
